# CurseForge integration status

CurseForge support is deferred to the `0.7.x` development line.

MCW Launcher does not publish or embed the approved CurseForge API key in public `0.6.x` builds. The developer is requesting written clarification from Overwolf about the permitted authentication architecture for a public desktop launcher.

The `0.6.x` line focuses on:

- Forge installation, repair, rollback, compatibility checks, and diagnostics;
- manual Fabric/Forge mod management;
- Fabric and Forge mods from Modrinth;
- Fabric and Forge `.mrpack` modpacks from Modrinth.

See [`FORGE_MODRINTH.md`](FORGE_MODRINTH.md) for the supported Beta 5 workflow.
